# 🔠 Uppercase Letter Checker (C++)

This C++ program reads a single character from the user and determines whether it is an **uppercase letter**.

---

## 🧠 Objective
To practice conditional statements, logical operators, and character handling in C++.

---

## 📜 Problem Statement
> Write a C++ program that reads a character and prints **"Uppercase Letter"** if it is an uppercase alphabet, otherwise prints **"Not an uppercase Letter"**.

---

## 💻 Code Overview
The program:
1. Prompts the user to enter a character.
2. Uses the `isupper()` function from `<cctype>` to check if the character is uppercase.
3. Adds an extra condition to ensure the character lies between `'A'` and `'Z'`.
4. Displays the result.
